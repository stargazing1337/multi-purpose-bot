ModBot
=================

Easily expandable, customizable and fully featured Discord moderation bot
___

Features  
------------


Information Commands

`help` `about` `ping` `weather` 
  
Moderation Commands

`ban` `kick` `prune` `prefix` `tempmute`  

User Commands

`report` `serverinfo` `userinfo`  

Fun Commands

`8ball` `coinflip`
